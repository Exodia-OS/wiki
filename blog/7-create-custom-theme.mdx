you can easily create your themes with the same steps for all WMs

- Go to `~/.config/bspwm/themes/`
- Copy an existing theme and rename it. (eg:`Etherium`)
- Edit `theme.conf`
  - ![](/img/tips/CreateCustomTheme/1.png)
- Generate Color Scheme for the theme using `python-pywal`
  - ```bash
    wal -i /usr/share/backgrounds/Etherium.jpg
    ```
  - ![](/img/tips/CreateCustomTheme/GeneratedColorScheme.png)
  - add the Color Scheme to `colorScheme`
  - ![](/img/tips/CreateCustomTheme/AddColorScheme.png)
  - :::note[Hint]
    you can use `colorPicker` to capture colors, press on <kbd>mod + p</kbd> to use it
    :::
- Then Edit `polybar` config
  - :::warning[]
    in the polybar modules update the script paths for all modules files
    ![](/img/tips/CreateCustomTheme/polybar-1.png)
    ![](/img/tips/CreateCustomTheme/polybar-2.png)
    ![](/img/tips/CreateCustomTheme/polybar-3.png)
    :::
- Then take a screenshot and save it in the theme directory with name `preview.png`
  - ![](/img/tips/CreateCustomTheme/2.png)

# Contributing

do you want to share your theme?

- Fork [**`bspwm-themes`**](https://github.com/Exodia-OS/bspwm-themes) Repo
- add your theme to the [**`community`**](https://github.com/Exodia-OS/bspwm-themes/tree/master/community-themes) directory
- Push your changes to a new branch with `dev-${GITHUB-USERNAME}`
  - ```bash
    ./git-push.sh -t "dev-mmsaeed509"
    ```
- Submit a pull request
