### Features?
- Comes with [**`BSPWM`**](https://github.com/baskerville/bspwm) pre-configured with 13+ fancy Themes
- [**`zsh`**](https://www.zsh.org/) (pre-installed) default shell.
- [**`omz`**](https://github.com/ohmyzsh/ohmyzsh) (pre-installed ,community-driven framework for managing) pre-configured with a lot of plugins like git, zsh-autosuggestions, zsh-syntax-highlighting .
- [**`powerlevel10k`**](https://github.com/romkatv/powerlevel10k) (pre-installed) the default omz theme .
- [**`NvChad`**](https://github.com/NvChad/NvChad) (pre-installed) neovim config written in lua that makes neovim looks like IDE, pre-configured with a lot of plugins, dashboard enabled.
- [**`ani-cli`**](https://github.com/pystardust/ani-cli) (pre-installed) for watching anime using terminal.
- [**`ytfzf`**](https://github.com/pystardust/ytfzf) (pre-installed) for watching YouTube videos using terminal.
- [**`predator-sense`**](https://github.com/Exodia-OS/predator-sense) (pre-installed) to control CPU/GPU Fan, Keyboard RGB(ONLY for Acer Predator Laptops).
- [**`Keybinding`**](https://exodia-os.github.io/exodia-website/Keybinding).
