## Changelog :-

- updated
 * Logo
 * GRUB
 * calamares slide show config
 * rofi themes
 * polybar themes
 *  `.zshrc` theme (added lsd)
 * updated ISO with latest PKGs
 * some config files in bspwm
 * updated Templates files (help you to create coding files like -> C, C++, web Dev, python, etc...)
 * updated `jgmenu`( added PowerShell)
 * updated `thunar` menus ( added -> open PowerShell here)

- new
 * Added PowerShell (fully customized)
 * Added `Exodia-TUI-Apps` mapscii, browsh, etc... to list all available apps run this command `pacman -Sg Exodia-TUI-Apps`
 * Added new theme
 * Added `jgmenu` (pre-configured)
 * Added `xdo` and `xqp` for `jgmenu` (To Toggle jgmenu while right click on desktop)
 * Added `EWW` pre-installed, pre-configured with `exodia-eww`, use <kbd>mod + d</kbd>  to open/close EWW dashboard
 * Added `plymouth`, `exodia-plymouth-theme`
